Online Bank Account Management System
--------------------------------------

How to Compile:
    javac OnlineBankSystem.java

How to Run:
    java OnlineBankSystem

Features:
- Create account
- Deposit money
- Withdraw money
- View balance
- Proper exception handling:
    * Invalid deposit amount (InvalidAmountException)
    * Withdraw more than balance (InsufficientFundsException)
    * InputMismatchException for invalid input
    * General Exception handling
    * Scanner closure in finally block
